# 福岡六天五夜完整攻略

A single-file HTML travel guide for a 6-day / 5-night trip to Fukuoka, Japan (Dec 25–30).

**Live site:** https://e9971217.github.io/FukTrip/

## Itinerary

| Day | Date | Theme |
|-----|------|-------|
| Day 1 | 12/25 (Fri) | Arrive Fukuoka · Check-in · Yakuin Yakiniku · Yatai |
| Day 2 | 12/26 (Sat) | Karato Market · Mojiko Retro · THE OUTLETS |
| Day 3 | 12/27 (Sun) | Yufuin Onsen · Kinrinko · Beppu Check-in |
| Day 4 | 12/28 (Mon) | Dazaifu Tenmangu · Umegae Mochi · Back to Hakata |
| Day 5 | 12/29 (Tue) | Takachiho Gorge · KKday Full-day Tour |
| Day 6 | 12/30 (Wed) | Kushida Shrine · Shopping · Fly Home |

## Features

- Fully offline-capable single HTML file
- Responsive layout for iPhone / mobile
- Sticky navigation bar with day links
- Timeline-based daily itinerary
- Google Maps links for key restaurants
- Airport transfer photo guide (Day 1)
- Flight info, accommodation, and booking checklist sections
